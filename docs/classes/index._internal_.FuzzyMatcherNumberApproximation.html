Class FuzzyMatcherNumberApproximation

The instance of this fuzzy matcher will match any number, which is approximately equal to the expected value, with the notion of "approximate equality" defined by the approx configuration property.

This class is usually instantiated with the helper method Test.anyNumberApprox

For example:


t.is(10.5, t.anyNumberApprox(10))

t.is(10.1, t.anyNumberApprox(10, { percent : 1 }))
